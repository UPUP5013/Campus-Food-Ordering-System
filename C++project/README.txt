# 校园点餐系统 (Campus Food Ordering System)

## 项目简介
这是一个基于C++控制台开发的校园点餐管理系统。项目旨在解决校园内商家信息分散、点餐流程混乱的问题，将用户、商家、外送员三方整合到一个统一的平台中，实现从下单到配送的完整闭环。

## 核心功能
- **用户模块**：注册、登录、浏览商家、下单、查看/取消订单。
- **商家模块**：查看待处理订单、接单、标记出餐、管理菜单。
- **外送员模块**：注册与身份验证、查看待取餐订单、接单配送、确认送达。
- **数据持久化**：所有数据通过文件系统（.txt）读写，不依赖外部数据库。
- **订单状态流转**：完整实现 `pending → cooking → ready → delivering → completed` 的状态流转。
- **安全机制**：包含外送员身份验证、订单归属校验等安全防护。
- **全流程退出机制**：所有输入界面支持输入 `0` 安全取消并返回上级菜单。

## 技术亮点
- 面向对象设计，将 `User`、`Merchant`、`DeliveryPerson` 抽象为独立模块。
- 分层架构：数据层（FileManager）与业务层分离，降低模块间耦合。
- 继承与多态：`Merchant` 基类与 `DeliveryPerson` 基类的设计为未来扩展预留空间。
- 健壮性处理：独立诊断并解决了栈溢出（递归→循环）、中文编码乱码、文件换行符缺失、订单权限漏洞等问题。

## 项目结构
